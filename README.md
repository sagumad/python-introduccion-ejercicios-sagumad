## Descripción del trabajo

Este trabajo consiste en una serie de 10 ejercicios prácticos realizados en Python. El objetivo es aplicar los conceptos básicos vistos en clase durante el tercer semestre en la materia Nuevas Tecnologías de Programación.

El lenguaje nuevo que estamos aprendiendo es Python, y en este taller se trabaja su parte más básica, enfocándonos en entender la lógica de programación y el uso correcto de sus estructuras fundamentales.


## Objetivos

* Comprender el uso de la función `print()`.
* Declarar y usar variables.
* Aplicar operadores aritméticos.
* Implementar estructuras condicionales.
* Utilizar ciclos `for` y `while`.
* Trabajar con las estructuras de datos fundamentales de Python.
* Solicitar datos al usuario con `input()`.
* Crear y utilizar funciones.


## Contenido del proyecto

El proyecto contiene los siguientes archivos:

* `ejercicio_01.py` → Uso básico de print.
* `ejercicio_02.py` → Declaración y uso de variables.
* `ejercicio_03.py` → Operadores aritméticos.
* `ejercicio_04.py` → Condicionales.
* `ejercicio_05.py` → Ciclos for y while.
* `ejercicio_06.py` → Estructuras de datos (lista, tupla, conjunto y diccionario).
* `ejercicio_07.py` → Entrada de datos y conversión de tipos.
* `ejercicio_08.py` → Iteración de colecciones.
* `ejercicio_09.py` → Formateo de texto e interacción con el usuario.
* `ejercicio_10.py` → Creación y uso de funciones.

Cada archivo desarrolla un tema específico visto en clase.


## Cómo ejecutar los archivos

1. Tener Python instalado en el computador.
2. Abrir una terminal o consola.
3. Ubicarse en la carpeta donde están los archivos.
4. Ejecutar el archivo con el comando:

```
python nombre_del_archivo.py
```

Ejemplo:

```
python ejercicio_01.py
```
